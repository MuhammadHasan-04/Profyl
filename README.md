# Profyl – Resume Builder

Profyl is a modern, responsive Resume Builder web application that allows users to create, preview, and customize professional resumes in real-time.

Built with a clean UI and modular architecture, Profyl provides multiple templates and a live preview system for seamless resume generation.

---

## Features

- Multiple Resume Templates (Default & Clean)
- Live Resume Preview
- Fully Responsive Design (Mobile, Tablet, Desktop)
- Modular Component Structure
- Modern UI with TailwindCSS
- Dynamic Data Rendering
- Organized Folder Structure
- Scalable Architecture

---

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Lucide React
- React Router DOM

### Backend
- Node.js
- Express.js
- SQLite
